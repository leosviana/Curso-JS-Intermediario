PROPRIEDADES E MÉTODOS DE ARRAY

SORT()

Classifica os elementos de um array em ordem crescente.

PROFISSOES: [0]Engenheiro, [1]Pedreiro, [2]Balconista, [3]Faxineiro, [4]Motorista

REVERSE()

Inverte a ordem de um array de trás para frente

PROFISSOES: [0]Engenheiro, [1]Pedreiro, [2]Balconista, [3]Faxineiro, [4]Motorista

ORDEM DESCRESCENTE USANDO SORT() E REVERSE()

Classifica os elementos de um array em ordem decrescente.

Primeiro se usa o método sort() para os elementos ficarem em ordem crescente.

Em seguida, utiliza o método reverse() para inverter a ordem de todos os elementos.

PROFISSOES: [0]Engenheiro, [1]Pedreiro, [2]Balconista, [3]Faxineiro, [4]Motorista

PUSH()

Adiciona novos elementos ao final de um array e retorna o novo comprimento.

ANIMAIS: [0]Macaco, [1]Leão, [2]Elefante -> Adiciona [3]Hipopótamo

POP()

Remove o último elemento de um array e retorna esse elemento.

ANIMAIS: [0]Macaco, [1]Leão, [2]Elefante, [3]Hipopótamo

CONCAT()

Une um ou mais arrays. Retorna um novo array contendo os arrays unidos.

ARRAY 1 - CAMISAS: [0]Manga Larga, [1]Cavada

ARRAY 2 - CALÇADOS: [0]Sapatenis, [1]Botina, [2]Chinelo

SHIFT()

Remove o primeiro elemento de um array e retorna esse elemento.

CORES: [0]Branco, [1]Vermelho, [2]Azul

UNSHIFT()

Adiciona novos elementos ao início de um array e retorna o novo comprimento.

CORES: [0]Branco, [1]Vermelho, [2]Azul

Adicionar as cores: PRETO, AMARELO

SLICE()

Seleciona uma parte de um array e retorna o novo array (Inicio para o fim).

PIZZA: [0]queijo, [1]pepperoni, [2]vegetais, [3]azeitonas, [4]Presunto, [5]Ovos

SLICE() Ao contrário

Seleciona uma parte de um array e retorna o novo array (Fim para o inicio).

PIZZA: [0]queijo, [1]pepperoni, [2]vegetais, [3]azeitonas, [4]Presunto, [5]Ovos, [6]Orégano

SPLICE()

Adiciona ou remove elementos do array.

ITENS: [0]poção de cura, [1]poção de magia, [2]armadura, [3]escudo

Adiciona dois elementos: espada, sapatos.

NÚMEROS: 1, 2, 3, 4, 5

FILTER()

O método filter() cria um novo array preenchido com elementos que passam em um teste fornecido por uma função. Não executa a função para elementos vazios. Não altera o array original.

IDADE: 15, 16, 17, 18, 19, 20, 21

Retorna a idade maior que 18

REDUCE() - Exemplo 1

Reduz os valores de um array para um único valor (da esquerda para a direita).

NUMEROS: 175, 50, 25

Retorna a dedução dos valores

REDUCE() - Exemplo 2

Soma todos os valores do array e arredonda no final.

NUMEROS: 15.5, 2.3, 1.1, 4.7
